Tecnologias
CKEditor 5 Bootstrap 5 JavaScript ES6 LocalStorage API

O Cronograma de Estudos v2.2 foi projetado para ser uma ferramenta de organização offline robusta, intuitiva e totalmente personalizável.

Gestão de Matérias
  • Múltiplas Matérias: Crie, renomeie e gerencie cronogramas independentes em uma única interface.
  • Alternância Rápida: Troque de matéria instantaneamente; o sistema lembra o último acesso.
Edição e Personalização
  • Edição Rica (CKEditor 5): Suporte a negrito, cores, listas e links nas observações.
  • Colunas Personalizáveis: Renomeie os cabeçalhos das colunas diretamente na tabela.
  • Visibilidade Seletiva: Oculte colunas que você não utiliza pelo painel de preferências.
Automação e Produtividade
  • Cálculo de Datas: Sugestão automática da próxima data baseada no intervalo configurado.
  • Atalhos de Teclado: Use Ctrl + S para salvar instantaneamente.
  • Auto-Save: Sistema de salvamento automático a cada 5 minutos.
  • Prevenção de Perda: Alerta visual e aviso do navegador ao tentar fechar sem salvar.
Dados e Portabilidade
  • Privacidade Total: Dados salvos localmente (LocalStorage). Nada sai do seu PC.
  • Backup: Exporte e restaure seu progresso em arquivos .json.
  • Mesclagem Inteligente: Escolha entre substituir tudo ou apenas adicionar dados novos.
Saída e Layout
  • Modo Impressão: Layout otimizado para papel A4 (paisagem) ou PDF.
  • Design Moderno: Baseado em Bootstrap 5 para máxima legibilidade.
